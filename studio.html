The Stack: Foundation models. Federated training. Edge inference.

Physics-informed neural nets fine-tuned for emerging-market constraints. All numbers trace to production.

Foundation Models

Vision Transformers — SegFormer-B2 • Swin-v2-Base • DINO-v2 (semantic segmentation). Pre-trained on ImageNet-22k, fine-tuned on 6-band multispectral + SAR composites. Physics layers enforce energy conservation (SegFormer + LSTM-based flow models).

Federated Training

On-device model updates (TensorFlow Lite, ONNX Runtime) encrypted via Paillier homomorphic schemes. Aggregates gradient contributions from 50+ edge nodes without centralising sensitive data. Deployed on Raspberry Pi 4B, Jetson Nano (4 GB), commodity Android (8 GB RAM).

Edge Inference

<120 ms latency (P95) for 512×512 SAR segmentation on Pi 4B. Quantised INT8 models (post-training, QAT). Fallback to cloud (AWS Lambda GPU) when edge compute saturates. Offline-first: stores 7 days of predictions locally, syncs when bandwidth available.
